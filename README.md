# 📂 Tasks Included

## Task 1 — Population Data Visualization
Created bar charts and histograms to visualize categorical and continuous variables such as age and gender distribution.

## Task 2 — Exploratory Data Analysis (EDA)
Performed data cleaning and exploratory data analysis on datasets like the Titanic dataset to identify patterns and relationships.

## Task 3 — Decision Tree Classifier
Built a Decision Tree Machine Learning model to predict customer purchasing behavior using the Bank Marketing dataset.

## Task 4 — Sentiment Analysis
Analyzed and visualized sentiment patterns in social media data to understand public opinions and attitudes.

## Task 5 — Traffic Accident Analysis
Analyzed traffic accident datasets to identify trends related to weather, road conditions, accident hotspots, and contributing factors.
